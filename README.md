# resto-project

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### json server install & start (handle backend)
```
npm install -g json-server && json-server --watch json-server
```
